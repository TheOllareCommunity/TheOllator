## Instructions
### Frontend
Run ```pip install simpleaudio``` from terminal if not playing

### Backend
Run first ```SET FLASK_APP=backend.py``` (**win**) or ```export FLASK_APP=backend.py``` (**mac**) and then ```flask run --port=8080``` from terminal to boot server. Alternatively ```python backend.py``` simply also works and the script automatically sets the environment variables.
